# Nguyễn Huy Hoàng - Portfolio

A portfolio website showcasing my experience, skills, and projects as a Frontend Developer.

## Tech Stack

### Framework

- **Next.js 15** - React framework with server-side rendering and routing
- **React 19** - UI component library

### UI & Styling

- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Accessible UI components built with Radix UI and Tailwind
- **Lucide React** - Beautiful & consistent icon set
- **Framer Motion** - Animation library for React

### Form Handling & Validation

- **React Hook Form** - Form state management and validation
- **Zod** - TypeScript-first schema validation

### Quality & Development

- **TypeScript** - Static type checking
- **ESLint** - Code quality and consistency

### Features

- Responsive design for all device sizes
- Interactive UI components and animations
- Dark theme optimized design
- Animated counters and transitions
- Project filtering and categorization
- Contact form functionality

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm run dev

# Build for production
pnpm run build

# Start production server
pnpm start
```
